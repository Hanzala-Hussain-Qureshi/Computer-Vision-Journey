# Real-Time Face Detection

A computer vision project that detects human faces 
in real-time using a webcam and OpenCV.

---

## 🎯 What It Does
- Opens your webcam
- Detects faces in real-time
- Draws a green rectangle around each face detected

---

## 🛠️ Technologies Used
- Python 3.x
- OpenCV

---

## ⚙️ How To Run

1. Clone this repository
```
git clone https://github.com/Hanzala-Hussain-Qureshi/Computer-Vision-Journey
```

2. Install the required library in Terminal
```
pip install opencv-python
```

3. Run the program
```
python Face_Detection.py
```

4. Press **Q** to quit
